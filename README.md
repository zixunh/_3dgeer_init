<h1>gsplat-geer: An Open-Source Library for Exact and Efficient Gaussian Rendering</h1>

This repository contains the `gsplat-geer` extension implementation based on the paper "3DGEER: 3D Gaussian Rendering Made Exact and Efficient for Generic Cameras", which can be found [here](https://arxiv.org/abs/2505.24053).

## Key features:
fixed grid-line artifacts in mismatched association of 3dgut, rendering use the heuristic max response following 3dgut in gsplat; see mathmatical provement in 3dgeer.
